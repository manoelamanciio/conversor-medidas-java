# 📏 Conversor de Medidas (Java)

Projeto em Java que converte uma distância em metros para outras unidades do sistema métrico.

## 🚀 Funcionalidades
- Conversão para km, hm, dam, dm, cm e mm
- Menu interativo no terminal
- Execução contínua até o usuário sair

## 🛠️ Tecnologias
- Java
- Scanner
- Estruturas de repetição e condição
