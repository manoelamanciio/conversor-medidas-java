# Conversor de Medidas (Java)

Um projeto em **Java** que converte uma distância em **metros** para outras unidades do sistema métrico. Originalmente o exercício pedia apenas as conversões, mas eu implementei um **menu interativo** que possibilita ao usuário escolher qual unidade deseja converter. Além disso, este projeto também serviu para eu treinar **debug**, lógica de programação e uso de estruturas condicionais e de repetição em Java.

---

## 🧠 Funcionalidades

✔️ Converte metros para as seguintes unidades:
- Quilômetros (km)  
- Hectômetros (hm)  
- Decâmetros (dam)  
- Decímetros (dm)  
- Centímetros (cm)  
- Milímetros (mm)

✔️ Menu interativo no terminal para seleção da unidade desejada  
✔️ Ciclo de execução contínuo até o usuário optar por sair do programa  
✔️ Implementação orientada à lógica e boas práticas em Java  

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com foco em:</br>
- **Java** – linguagem principal do projeto  
- **Scanner** – para entrada de dados via teclado  
- **Estruturas condicionais e de repetição** – para lógica de seleção e ciclo do programa

## ▶️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/manoelamanciio/conversor-medidas-java.git
  
2. Abra o projeto em sua IDE favorita (Ex: IntelliJ, VS Code ou Eclipse).

3. Compile e execute a classe principal (Main.java):
   javac Main.java
java Main

4. O programa pedirá que você insira um valor em metros e selecione a unidade para conversão.
